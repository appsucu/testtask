# NovaClean Eco — виконання ТЗ

Короткий звіт по реалізації лендингу хімчистки.

## Вимоги

- ✅ **Адаптивний дизайн**: реалізовано брейкпоінти `1024px / 768px / 480px` та адаптацію ключових блоків (header, hero, timeline, contact, footer).
- ✅ **Форми — заглушка**: форма контактів має `data-stub="true"`, обробляється у `js/main.js` без бекенду (toast-повідомлення).
- ✅ **Meta tags**: додані `description`, `keywords`, `author`, `robots`, Open Graph (`og:title`, `og:description`, `og:type`, `og:locale`), `viewport`.
- ✅ **Cookie notification**: банер cookies з кнопками прийняття/відхилення та збереженням вибору в `localStorage`.
- ✅ **Header + Footer**: повністю реалізовані, включно з навігацією, контактами та юридичними посиланнями.

## Додатково

- ✅ **Сучасний дизайн**: оновлений hero (full-width background, затемнення, декоративні еко-акценти), анімації, стилізована карта.
- ✅ **Логічна структура**: окремі секції Home / About / Pricing / Blog / Contacts / Policy з hash-навігацією.
- ✅ **Тематичний контент**: контент приведено до концепції еко-хімчистки майбутнього (без згадок AI/ШІ у текстах).

## Результат

- ✅ **Формат**: готовий проєкт у репозиторії (`Task2/`) для публікації або архівації.
